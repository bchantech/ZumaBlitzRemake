maps/*/config.json

This is the map configuration file. Defines the Map's name, paths and sprites.

  • The root entry.
    • name: The map name.
    • paths: A list of paths which are available on this map.
      • A single path, which is a list of nodes.
        • A single path node.
          • x: The X position of this node.
          • y: The Y position of this node.
          • hidden: When set, the segment from this to next node will be hidden - the spheres will become darker, unshootable, and they will be rendered behind non-background sprites.
          • warp: When set, the spheres will instantly teleport to the next node once they reach this node.
    • sprites: A list of sprites to be drawn on this map.
      • A single sprite.
        • x: The X position of this Sprite.
        • y: The Y position of this Sprite.
        • path: The path to the Sprite to be displayed.
        • internal: When set, the path field will refer to this map's folder as the base directory. Otherwise, the base directory will be the root game folder.
        • background: If set, this sprite will be rendered behind all spheres. Otherwise, it will be covering hidden spheres.
