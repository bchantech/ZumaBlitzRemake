config/hud_layer_order.json

This file lists all UI layers and the order in which they will be drawn, in order from the bottom to the top.

All gameplay elements are rendered first and thus are displayed at the bottom, sometimes with a MAIN layer assigned.

The first layer specified in this file should be "MAIN"!

  • A list of layers.
    • A single layer name.
