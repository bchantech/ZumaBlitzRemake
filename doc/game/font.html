fonts/*.json

Defines a single Font.

image

An image will be used as a source for this Font.

  • The root object.
    • type: "image"
    • image: The image file containing all of the Font's characters.
    • characters: Defines all characters. Keys are single characters.
      • offset: The horizontal offset where the character starts, in pixels.
      • width: The width of this character, in pixels.

truetype

A TTF file will be used as a source for this Font.

  • The root object.
    • type: "truetype"
    • path: A font file to be used for this Font.
    • size: The size of this Font, in pixels.
    • color: The color of this Font.
