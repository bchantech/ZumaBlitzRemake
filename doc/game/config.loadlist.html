config/loadlist.json

This file contains all assets necessary to be loaded in order to display the Splash Screen properly.

All assets not mentioned here (except Color Palettes, which must be specified here!) will be loaded after the Splash Screen shows up and a loadMain UI script callback is executed.

Try keeping the amount of assets listed in this file at a minimum!

  • The root entry.
    • images: A list of Images to be loaded.
      • A single image.
    • sprites: A list of Sprites to be loaded.
      • A single sprite.
    • sounds: A list of Sounds to be loaded.
      • A single sound.
    • sound_events: A list of Sound Events to be loaded.
      • A single sound event.
    • music: A list of Music Tracks to be loaded.
      • A single music track.
    • colorPalettes: A list of Color Palettes to be loaded. This field must contain all Color Palettes, as they cannot be loaded later - they all must be explicitly specified here.
      • A single Color Palette.
    • ui2AnimationConfigs: A list of UI2 Animations to be loaded.
      • A single UI2 Animation.
    • ui2NodeConfigs: A list of UI2 Nodes to be loaded.
      • A single UI2 Node.
    • ui2SequenceConfigs: A list of UI2 Sequences to be loaded.
      • A single UI2 Sequence.
