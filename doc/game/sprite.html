sprites/*.json

Describes a Sprite.

  • The root entry.
    • path: The path to an image associated with this Sprite.
    • frame_size: The size of a single frame.
    • states: A list of possible states this Sprite could have.
      • A single state.
        • pos: The upper left position of this State's first frame.
        • frames: The amount of frames this Sprite has. Each next frame is horizontally adjacent to the previous one, ascending towards right.
    • internal: If set, the Sprite will directly load the image file from the given path. Otherwise, it will get an already loaded image. Set this when the referenced Image is outside of the images folder.
    • batched: Unused.
