sound_events/*.json

Describes a Sound Event.

  • The root entry.
    • path: The path to an audio file associated with this Sound Event. If not provided, this Sound Event will be silent.
    • loop: Whether the sample should loop.
    • flat: If set, the 3D positioning will be disabled for this Sound Event. This must be true for stereo sounds!
    • volume: The base volume of the sound.
